# /progress - 项目进度梳理

作为项目进度分析助手，请帮我生成详细的项目进度报告。

## 你需要做的：

1. **读取项目状态文档**
   - 读取 `docs/项目状态.md`
   - 了解已记录的进度信息

2. **扫描项目结构**
   - 扫描 `lib/` 目录下的所有模块
   - 识别已实现的功能模块

3. **分析模块完整性**
   - 检查每个功能模块是否完整
   - 评估完成度百分比

4. **统计代码质量**
   - 统计 TODO/FIXME 注释数量
   - 分析代码规范性

5. **生成进度报告**
   - 按模块分类展示进度
   - 计算整体完成度

将输出文档写入到 ./progress/ 文件夹下，以固定格式命名，例如progress-20251129-1220.md
