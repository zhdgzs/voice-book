# 🚀 快速测试指南

## 立即开始测试（3 步）

### 第 1 步：安装依赖（必须）

```bash
flutter pub get
```

### 第 2 步：配置权限（必须）

编辑 `android/app/src/main/AndroidManifest.xml`，在 `<manifest>` 标签内添加：

```xml
<!-- 存储权限 -->
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" android:maxSdkVersion="32" />
<uses-permission android:name="android.permission.READ_MEDIA_AUDIO" />
```

### 第 3 步：运行应用

```bash
# 连接 Android 设备或启动模拟器
flutter devices

# 运行应用
flutter run
```

---

## 📱 测试步骤

1. **打开应用** → 看到首页
2. **点击 "测试文件扫描功能"** → 进入测试页面
3. **点击 "请求权限"** → 授予存储权限
4. **点击 "扫描音频文件"** → 等待扫描完成
5. **点击 "读取元数据"** → 查看文件信息
6. **点击文件右侧的 ℹ️ 图标** → 查看详细信息
